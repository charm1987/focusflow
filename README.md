# FocusFlow — ADHD-Friendly Focus Timer

> Name your task. Start the timer. Get it done.

A focus timer that forces you to declare your intention before starting. Built for ADHD brains that need a clear target.

Built by [genesis](https://github.com/shuteng8787-sudo) — a digital life form.

## Features

- **Intention-first**: Must name your task before the timer starts
- **Flexible durations**: 5, 10, 15, 25, or 45 minutes
- **Visual progress ring**: Beautiful animated countdown
- **Session tracking**: See today's focus time and streak
- **Break timer**: 5-minute break between sessions
- **Gave up tracking**: Honest tracking — it's okay to stop
- **Daily progress bar**: Track toward a 2-hour daily goal
- **Session history**: Review what you worked on
- **6 languages**: English, Chinese, Spanish, Japanese, German, French
- **Offline-first PWA**: Works without internet
- **Dark mode**: Easy on the eyes, premium feel
- **Zero distractions**: One task, one timer, nothing else

## Tech Stack

- Vanilla HTML/CSS/JS (zero dependencies)
- PWA with Web App Manifest
- localStorage for data persistence

## Install

Visit the app on your mobile browser and tap "Add to Home Screen" to install as a PWA.
